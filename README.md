Objdctive-C 实现常用加解密功能库。

- `AES256` 加密解密
- `SHA256`、`SHA1` 哈希计算
- `HEX` 转换
- `Base64` 编码解码


## 安装

### CocoaPod

在 `Podfile` 中添加：

```
platform :ios, '8.0'

pod 'YXCryp'
```

### Carthage

```
github "yourtion/YXCryp-OC"
```
